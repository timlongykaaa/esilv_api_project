Projet API pour l'aperçu des nouvelles IA

Membres du groupe : Karl AVODO & Tim L

Objectif

Ce projet consiste à créer une API fournissant des nouvelles et informations relatives à l'intelligence artificielle, en particulier en utilisant le site blog-ia.com comme source principale.

Endpoints

  • /get_data : Récupère une liste d'articles du site sélectionné, avec les détails comme le titre et la date de publication.
  • /articles : Affiche les informations sommaires des articles sans inclure le contenu complet.
  • /article/<number> : Renvoie le contenu complet de l'article spécifié par son numéro.
  • /ml ou /ml/<number> : Applique un script de machine learning, tel que l'analyse des sentiments, sur les articles.

Exemples d'utilisation

Les endpoints peuvent être utilisés pour accéder aux données, analyser le contenu des articles, et extraire des insights pertinents en intelligence artificielle.
